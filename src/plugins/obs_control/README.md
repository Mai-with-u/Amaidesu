# 🎥 OBS 控制插件

将弹幕、聊天等消息实时推送至 OBS Studio，实现直播画面中的文字弹幕上屏功能。

## ✅ 功能特性

- 实时将接收到的消息（如 B站弹幕）显示在 OBS 文本源中  
- 支持自定义 OBS 连接参数（主机、端口、密码、文本源名称）  
- 自动重连机制，网络波动后自动恢复  
- 支持其他插件调用 `send_to_obs()` 方法直接发送文本  
- 全中文日志输出，便于调试和维护

## ⚙️ 依赖要求

- `obsws-python`：用于与 OBS WebSocket 通信

安装命令：
```bash
pip install obsws-python
